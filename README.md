## Iniciando o projeto

Primeiramente será necessário clonar este projeto em (https://github.com/BrunoCarvalhoFeitosa/gta-vi.git), após o download será necessário abrir este projeto no seu editor e no terminal digitar npm install ou yarn, posteriormente é só rodar em seu terminal o comando npm run dev, após isso, a página será aberta em seu navegador.

### Pré-requisitos

* npm
  ```sh
  npm install npm@latest -g
  ```

### Instalação

1. Clone o repositório
   ```sh
   https://github.com/BrunoCarvalhoFeitosa/gta-vi.git
   ```
2. Instale os pacotes do NPM
   ```sh
   npm install
   ```
   
3. Inicie o projeto
   ```sh
   npm run dev
   ```
